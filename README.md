# 📰 Berita Portal – React + Vite

Portal berita dinamis berbasis **React**, **Vite**, dan **TailwindCSS** yang menampilkan berita terbaru dari berbagai kategori secara real-time menggunakan API dari [seal-berita-terkini](https://seal-berita-terkini.vercel.app/).

---

## ✨ Fitur

- 🔥 Berita dinamis dari berbagai kategori
- 📰 Headline, berita populer, rekomendasi, dan detail berita
- ⚡ Single Page Application (SPA) dengan React Router
- 🎨 UI modern dan responsif menggunakan TailwindCSS
- 💬 Sistem komentar lokal menggunakan `localStorage`
- 🧭 Breadcrumb navigation untuk pengalaman pengguna yang lebih baik
- 🎞️ Slider berita dan fitur UX lainnya

---

## 🚀 Instalasi & Penggunaan

### 1. Clone Repository

```bash
git clone https://github.com/raiakmal/seal-berita-terkini.git
cd seal-berita-terkini
```

### 2. Install Dependencies

Menggunakan **npm**:

```bash
npm install
```

Atau menggunakan **yarn**:

```bash
yarn install
```

---

### 3. Jalankan Development Server

Menggunakan **npm**:

```bash
npm run dev
```

Atau menggunakan **yarn**:

```bash
yarn dev
```

Aplikasi akan berjalan di:

```bash
http://localhost:5173
```

Atau port lain yang muncul pada terminal.

## 📂 Struktur Fitur

- **Home** → Menampilkan headline dan berita terbaru
- **Kategori** → Berita berdasarkan kategori tertentu
- **Detail Berita** → Halaman detail artikel
- **Komentar** → Penyimpanan komentar menggunakan `localStorage`
- **Responsive Layout** → Tampilan optimal di desktop maupun mobile
